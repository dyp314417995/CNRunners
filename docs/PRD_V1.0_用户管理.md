# 九州战马联盟 V1.0 PRD

## 1. 产品概述

### 产品名称
九州战马联盟（CNRunners）

### 产品定位
帮助跑团主理人高效管理跑团成员的小程序工具

### 核心价值
简化跑团成员管理流程，实现快速注册、群组管理和管理员体系

---

## 2. 用户角色

| 角色 | 说明 | 权限范围 |
|------|------|----------|
| 超级管理员（超管） | 跑团创始人/主理人 | 全部功能、添加/删除管理员、管理所有成员 |
| 管理员 | 跑团核心管理层 | 管理普通成员、审核积分等 |
| 普通成员 | 跑团成员 | 查看信息、提交跑步积分等 |

---

## 3. 功能需求

### 3.1 用户登录即注册

#### 触发时机
- 微信授权后首次使用小程序

#### 采集信息
| 字段 | 来源 | 必填 |
|------|------|------|
| 微信昵称 | 微信授权 | 是 |
| 微信头像 | 微信授权 | 是 |
| 手机号 | 用户输入 | 是 |
| 姓名 | 用户输入 | 是 |
| 微信号 | 用户输入 | 否 |
| 所属群组 | 用户选择 | 是 |

#### 群组选项
- 一群（默认）
- 二群（默认）
- 分舵群（管理员可配置启用/禁用）

#### 注册流程
```
微信授权 → 获取昵称/头像 → 输入手机号 → 输入姓名 → 选择群组 → 完成注册
```

---

### 3.2 管理员管理（超管专属）

#### 功能列表
| 功能 | 说明 |
|------|------|
| 添加管理员 | 指定普通成员为管理员 |
| 撤销管理员 | 管理员降级为普通成员 |
| 查看管理员列表 | 展示所有管理员信息 |
| 启用/禁用分舵群 | 控制分舵群选项是否可选 |

#### 管理员列表展示
- 头像
- 姓名
- 成为管理员时间
- 操作（撤销）

#### 操作限制
- 超管不可被撤销
- 至少保留一位超管

---

### 3.3 成员管理

#### 成员列表
- 支持按群组筛选
- 支持按姓名/手机号搜索
- 展示：头像、姓名、手机号、群组、注册时间

#### 成员详情
- 查看完整信息
- 修改群组
- 提升为管理员（普通成员时显示）
- 删除成员

---

### 3.4 超管身份验证

#### 验证方式
- 首次进入管理功能时，需验证超管身份
- 验证方式：输入超管密码（6位数字）
- 密码在首次使用时设置

## 4. 页面结构

### 4.1 页面清单

| 页面路径 | 页面名称 | 权限 |
|----------|----------|------|
| pages/public/login | 登录注册页 | 公开 |
| pages/index/index | 首页 | 登录用户 |
| pages/profile/index | 我的 | 登录用户 |
| pages/profile/members | 成员管理 | 管理员+ |
| pages/profile/settings | 设置 | 登录用户 |
| pages/admin/adminManage | 管理员管理 | 超管 |

### 4.2 页面关系图

```
登录注册页
    ↓
首页（TabBar）
    ├── 积分（TabBar）
    ├── 跑步（TabBar）
    └── 我的（TabBar）
            ├── 成员管理
            ├── 管理员管理（超管）
            └── 设置
```

## 5. 数据模型

### 5.1 用户表 (users)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | string | 用户唯一ID |
| nickname | string | 微信昵称 |
| avatar | string | 微信头像URL |
| phone | string | 手机号 |
| name | string | 姓名 |
| wechatId | string | 微信号 |
| groupId | string | 群组ID |
| role | string | super_admin / admin / member |
| createdAt | timestamp | 注册时间 |
| updatedAt | timestamp | 更新时间 |

### 5.2 群组表 (groups)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | string | 群组ID |
| name | string | 群组名称 |
| isEnabled | boolean | 是否启用 |
| sort | number | 排序 |
| createdAt | timestamp | 创建时间 |

### 5.3 初始群组数据

| ID | 名称 | 默认启用 |
|----|------|----------|
| group_1 | 一群 | 是 |
| group_2 | 二群 | 是 |
| group_3 | 分舵群 | 否（可配置） |

## 7. UI/UX 说明

### 7.1 登录注册页
- 顶部：跑团logo/名称
- 中部：微信头像预览区域（授权后显示）
- 手机号输入：带格式校验
- 姓名输入：纯中文/英文
- 群组选择：单选卡片式

### 7.2 成员管理页
- 顶部：搜索框 + 群组筛选
- 列表：头像 + 姓名 + 手机号 + 群组标签
- 列表项点击：进入详情
- 底部：成员统计

### 7.3 管理员管理页
- 顶部：Tab切换（管理员列表/群组设置）
- 管理员列表：头像 + 姓名 + 时间 + 撤销按钮
- 群组设置：开关控制分舵群

## 8. 异常处理

| 场景 | 处理方式 |
|------|----------|
| 微信授权失败 | 提示用户重试 |
| 手机号格式错误 | 实时校验，红色提示 |
| 群组未选择 | 高亮提示必选 |
| 删除唯一超管 | 阻止操作，提示"至少保留一位超管" |
| 超管密码错误 | 提示错误，3次锁定30分钟 |
