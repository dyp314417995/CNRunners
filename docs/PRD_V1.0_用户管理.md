# 九州战马联盟 V1.0 PRD

## 1. 产品概述

### 产品名称
九州战马联盟（CNRunners）

### 产品定位
帮助跑团主理人高效管理跑团成员的小程序工具

### 核心价值
简化跑团成员管理流程，实现快速注册、群组管理、积分管理和管理员体系

---

## 2. 用户角色

| 角色 | 说明 | 权限范围 |
|------|------|----------|
| 超级管理员（超管） | 跑团创始人/主理人 | 全部功能、添加/删除管理员、管理所有成员、管理积分分类 |
| 管理员 | 跑团核心管理层 | 管理普通成员、审核积分、录入积分、手动积分消耗 |
| 普通成员 | 跑团成员 | 查看信息、提交跑步积分、查看积分记录 |

## 3. 功能需求

### 3.1 用户登录即注册

#### 触发时机
- 微信授权后首次使用小程序

#### 采集信息
| 字段 | 来源 | 必填 |
|------|------|------|
| 微信昵称 | 微信授权 | 是 |
| 微信头像 | 微信授权 | 是 |
| 手机号 | 用户输入 | 是 |
| 姓名 | 用户输入 | 是 |
| 微信号 | 用户输入 | 否 |
| 城市 | 微信授权（省份/城市） | 否 |
| 所属群组 | 用户选择（支持多选） | 是 |

#### 群组选项
- 一群（默认启用）
- 二群（默认启用）
- 各分舵群（管理员可配置启用/禁用）
- 未加入（管理员可配置是否显示）

#### 群组选择规则
- 群组支持多选
- "未加入"与其他群组互斥：选了"未加入"则不能选其他群组，选了其他群组则不能选"未加入"

#### 城市选择
- 交互方式：省/市二级级联选择
- 第一级：选择省份（全国各省）
- 第二级：选择该省份下的城市
- 选择省份后，城市列表自动过滤为该省下的城市

#### 注册流程
```
微信授权 → 获取昵称/头像/城市 → 输入手机号 → 输入姓名 → 选择群组 → 完成注册
```

### 3.2 管理员管理（超管专属，不支持注册）

#### 功能列表
| 功能 | 说明 |
|------|------|
| 添加管理员 | 指定普通成员为管理员 |
| 撤销管理员 | 管理员降级为普通成员 |
| 查看管理员列表 | 展示所有管理员信息 |
| 启用/禁用群组 | 控制各群组选项是否可选 |
| 配置积分分类 | 增删改积分分类 |

#### 管理员列表展示
- 头像
- 姓名
- 成为管理员时间
- 操作（撤销）

#### 操作限制
- 超管不可被撤销
- 至少保留一位超管

---

### 3.3 成员管理

#### 成员列表
- 支持按群组筛选（多选）
- 支持按姓名/手机号搜索
- 支持按省份/城市筛选
- 展示：头像、姓名、手机号、群组、位置、注册时间

#### 成员详情
- 查看完整信息（姓名、手机号、微信号、省份/城市、群组）
- 修改群组
- 提升为管理员（普通成员时显示）
- 删除成员

---

### 3.4 积分管理

#### 3.4.1 积分分类（管理员可配置）

| 分类名称 | 说明 | 默认启用 |
|----------|------|----------|
| 拉新 | 推荐新成员加入 | 是 |
| 团服参赛 | 穿着团服参加比赛 | 是 |
| 自媒体 | 发布跑团相关内容 | 是 |
| 战队打卡 | 完成战队打卡任务 | 是 |
| 参加集体活动 | 参与跑团组织的活动 | 是 |

#### 3.4.2 自助积分提交流程

##### 用户操作
1. 选择积分类型
2. 填写积分说明（选填）
3. 上传图片凭证（必填，最多9张）
4. 提交等待审批

##### 管理员审批
1. 查看提交内容（图片+说明）
2. 审批通过 → 积分入账
3. 审批拒绝 → 用户可查看拒绝原因

##### 积分规则
- 积分有效期：每笔积分入账后1年内有效
- 积分过期后自动清零，不跨年累积

#### 3.4.3 管理员手动录入积分

##### 适用场景
- 用户无法自助提交的特殊情况
- 线下活动统一录入

##### 操作流程
1. 选择目标用户
2. 选择积分类型
3. 填写积分数量和原因
4. 确认录入

#### 3.4.4 积分消耗（管理员手动操作）

##### 适用场景
- 用户使用积分抵现消费
- 兑换跑团周边商品
- 其他抵现场景

##### 操作流程
1. 选择目标用户
2. 填写消耗积分数量
3. 填写消耗原因/用途
4. 确认消耗

---

### 3.5 超管身份验证

#### 验证方式
- 首次进入管理功能时，需验证超管身份
- 验证方式：输入超管密码（6位数字）
- 密码在首次使用时设置

## 12. 赛事日历功能（扩展模块）

### 12.1 功能概述

赛事日历为跑团成员提供赛事信息浏览、报名和个人赛事管理功能。

### 12.2 页面清单

| 页面路径 | 页面名称 | 说明 |
|----------|----------|------|
| pages/race/calendar | 赛事日历 | TabBar首页，日历+列表展示赛事 |
| pages/race/detail | 赛事详情 | 赛事信息、报名入口 |
| pages/race/register | 报名表单 | 填写报名信息 |
| pages/race/my | 我的赛事 | 关注的赛事、提醒设置 |

### 12.3 TabBar 结构

```
TabBar
├── 首页
├── 积分
├── 跑步
├── 赛事（新增）← TabBar
└── 我的
         └── 我的赛事（跳转）
```

### 12.4 功能详情

#### 12.4.1 赛事日历首页

- **日历视图**：月历展示，赛事日期显示小红点
- **列表视图**：赛事卡片展示
- **筛选功能**：全部/附近/省份/类型
- **赛事卡片**：封面图、名称、日期、地点、费用、报名状态

#### 12.4.2 赛事详情页

- **赛事信息**：时间、地点、规模、报名时间
- **竞赛项目**：分类选择（马拉松、半程、10公里等）
- **报名入口**：立即报名按钮
- **关注功能**：添加到我的赛事
- **分享功能**：分享给好友

#### 12.4.3 报名流程

```
选择项目 → 填写个人信息 → 阅读参赛声明 → 确认支付 → 报名成功
```

#### 12.4.4 我的赛事

- **关注列表**：用户关注的赛事
- **日历视图**：赛事日期标记
- **列表视图**：赛事卡片 + 提醒状态
- **提醒设置**：比赛当天/前一天/前三天/前一周
- **取消关注**：移除关注的赛事

- **跑团**：跑步爱好者自发组织的团体
- **超管**：超级管理员，跑团创始人/主理人
- **分舵群**：部分跑团在大群体下设的分小组
- **积分清零**：每笔积分入账满1年后自动清零
